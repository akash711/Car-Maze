# Car-Maze

C Program to find path from Start (S) to Destination (D)

- Takes input maze from user of max size 300
- Maze elements -> S (start), D (destination), # (block), . (legal move)
- Solution path marked by '+' symbol

<br>
Example input maze: <br>
S..#..# <br>
..##..# <br>
#..##.D <br>
#.....# <br>

<br>
Solution: <br>

S+.#..# <br>
.+##..# <br>
#++##+D <br>
#.++++# <br>
